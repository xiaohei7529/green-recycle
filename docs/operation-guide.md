## 1. 用户端操作指南

### 1.1 注册/登录

#### 方式一：手机号注册
1. 打开 APP/小程序
2. 点击"注册"
3. 输入手机号，获取验证码
4. 输入验证码，设置密码
5. 完成注册

#### 方式二：微信一键登录
1. 打开小程序
2. 点击"微信登录"
3. 授权获取用户信息
4. 自动完成注册/登录

### 1.2 扫码下单

**使用场景**: 家中有废品需要回收

**操作步骤**:
1. 打开 APP/小程序
2. 点击首页"扫码下单"
3. 扫描废品包装二维码（或手动选择废品类型）
4. 填写废品信息：
   - 废品类型（纸张/塑料/金属/电器等）
   - 预估重量
   - 上门时间
   - 详细地址
5. 提交订单
6. 等待回收员接单

**订单状态**:
- 🟡 待接单
- 🟠 已接单
- 🔵 回收中
- 🟢 已完成
- 🔴 已取消

### 1.3 预约上门回收

**操作步骤**:
1. 首页点击"预约回收"
2. 选择废品类型（可多选）
3. 填写预估重量
4. 选择上门时间（支持未来 7 天）
5. 确认地址（支持多个地址）
6. 提交预约

**时间选择**:
- 上午段：08:00-12:00
- 下午段：13:00-18:00
- 晚间段：19:00-21:00

### 1.4 查看附近回收点

**操作步骤**:
1. 首页点击"附近回收点"
2. 系统自动定位当前位置
3. 显示附近回收点列表
4. 点击回收点查看详情：
   - 地址
   - 营业时间
   - 回收价格
   - 用户评价
5. 导航前往

### 1.5 查看回收价格

**操作步骤**:
1. 底部导航点击"价格"
2. 查看各类废品实时价格
3. 价格分类：
   - 📰 纸类（报纸/纸箱/书本）
   - 🥤 塑料（PET/PE/PP）
   - 🥫 金属（铁/铝/铜）
   - 🔌 电器（大家电/小家电）
   - 👕 其他（衣物/玻璃）

**价格说明**:
- 价格每日更新
- 不同地区价格可能有差异
- 大宗回收可议价

### 1.6 回收记录

**操作步骤**:
1. 底部导航点击"我的"
2. 点击"回收记录"
3. 查看历史记录
4. 点击订单查看详情

**记录信息**:
- 订单编号
- 回收时间
- 废品类型
- 重量
- 金额
- 回收员信息
- 评价

### 1.7 积分系统

**获取积分**:
- ✅ 完成回收：1 元=1 积分
- ✅ 每日签到：+5 积分
- ✅ 邀请好友：+50 积分/人
- ✅ 完善信息：+20 积分

**积分使用**:
- 🎁 兑换礼品
- 💰 抵扣现金（100 积分=1 元）
- 🎫 兑换优惠券

### 1.8 消息通知

**通知类型**:
- 📦 订单状态变更
- 💰 收益到账
- 🎉 活动通知
- 📢 系统公告

**通知设置**:
1. "我的" → "设置" → "消息通知"
2. 开启/关闭各类通知

## 2. 回收员端操作指南

### 2.1 注册认证

**操作步骤**:
1. 下载回收员 APP
2. 点击"成为回收员"
3. 填写个人信息：
   - 姓名
   - 身份证号
   - 手机号
   - 服务区域
4. 上传证件：
   - 身份证正反面
   - 健康证（可选）
5. 等待审核（1-3 个工作日）
6. 审核通过，开始接单

### 2.2 接单流程

**操作步骤**:
1. 打开回收员 APP
2. 点击"订单" → "待接单"
3. 查看订单详情：
   - 用户地址
   - 废品类型
   - 预估重量
   - 预约时间
4. 点击"接单"
5. 联系用户确认

**接单技巧**:
- ⏰ 优先接距离近的订单
- 📦 优先接重量大的订单
- ⭐ 保持高评分，优先派单

### 2.3 上门回收

**操作步骤**:
1. 按预约时间上门
2. APP 点击"到达"打卡
3. 核实废品信息：
   - 实际类型
   - 实际重量
4. 确认金额
5. APP 点击"完成回收"
6. 用户确认支付

### 2.4 路线规划

**操作步骤**:
1. 点击"路线"
2. 查看今日订单分布
3. 点击"智能规划"
4. 系统生成最优路线
5. 按路线依次上门

### 2.5 收益统计

**查看方式**:
1. 点击"收益"
2. 查看收益报表：
   - 今日收益
   - 本周收益
   - 本月收益
   - 累计收益

**收益明细**:
- 订单收入
- 平台奖励
- 用户打赏
- 扣除项

**提现**:
1. 点击"提现"
2. 选择提现金额（≥100 元）
3. 选择提现方式（微信/支付宝/银行卡）
4. 提交申请
5. 1-3 个工作日到账

### 2.6 评价管理

**查看评价**:
1. 点击"我的" → "评价"
2. 查看用户评价
3. 回复评价（可选）

**提升评分**:
- ⏰ 准时上门
- 💬 礼貌沟通
- ⚖️ 公平称重
- 🧹 清理现场

## 4. 常见问题

### 4.1 用户常见问题

**Q: 如何修改上门地址？**
A: 下单后，在订单详情点击"修改地址"，回收员接单后无法修改。

**Q: 回收价格如何确定？**
A: 回收员上门后，根据实际废品类型和重量，按 APP 显示价格计算。

**Q: 如何取消订单？**
A: 回收员接单前可自助取消，接单后需联系回收员协商。

**Q: 回收员迟到怎么办？**
A: 可在订单页面点击"催单"，或联系客服处理。

**Q: 积分如何兑换？**
A: "我的" → "积分商城" → 选择商品 → 确认兑换。

### 4.2 回收员常见问题

**Q: 如何增加接单量？**
A: 保持高评分、高完成率，系统会优先派单。

**Q: 收益何时到账？**
A: 订单完成后即时到账，提现 1-3 个工作日。

**Q: 遇到恶意投诉怎么办？**
A: 在订单页面申诉，客服会核实处理。

**Q: 如何修改服务区域？**
A: "我的" → "服务区域" → 申请修改 → 审核通过。
